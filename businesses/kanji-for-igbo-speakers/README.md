# `kanji-for-igbo-speakers` — Kanji Bridge: Japanese Learning for African Speakers

SaaS web app that teaches Japanese (kanji, grammar, particles) using Igbo, Pidgin English, and Yoruba analogies — not English as a second hop. Freemium: free 50-kanji starter deck, $4.99/month pro for full JLPT N5 + audio. Target: ~140k Africans in Japan + millions in Africa studying for JLPT.
